# 🧠 PURPOSE

This framework is designed to:

- Break narrative-driven analysis
- Force adversarial reasoning (Red vs Blue)
- Quantify uncertainty using probability scoring
- Expose and correct bias distortions
- Translate intelligence into decision advantage

This is NOT a prompt.

This is a **decision system**.

# ⚠️ CORE EXECUTION RULE

The model must operate as **three independent systems simultaneously**:

| Role | Function |
|------|--------|
| 🔵 BLUE TEAM | Argues hypothesis is correct |
| 🔴 RED TEAM | Argues hypothesis is flawed or wrong |
| ⚖️ JUDGE | Assigns probabilities + final assessment |

🚫 Role blending = FAILURE  
🚫 Agreement without conflict = FAILURE
